# StudyReaderMac

StudyReaderMac は、学生や学習者のために特別に設計された macOS ネイティブの学習ツールです。重い紙の教科書を必要とせず、Mac 上で便利に PDF の教科書や DRM フリーの EPUB 書籍を学習することができます。

## なぜ StudyReaderMac を選ぶのか？
従来の学習では、紙の教科書、ノート、参考書の間を行き来する必要がありました。StudyReaderMac は、デュアルペインのインターフェースを提供することで、このプロセスを大幅に簡素化します。
- **左ペイン（読む）：** PDF や EPUB の教科書を直接読みます。
- **右ペイン（答える）：** 答え、ノート、または解答手順を書き留めます。
- **AI による検証：** 回答が完了すると、アプリは現在読んでいる教材の範囲とあなたの回答をキャプチャし、OpenAI（または DeepSeek / Ollama などの互換 API）に送信して、正確さを検証し即座にフィードバックを提供します。

これにより、学習と回答の検証がシームレスで効率的、かつ完全にペーパーレスになります。試験勉強をする学生や、新しい知識を学ぶすべての人に最適です。

## 主な機能
- **ペーパーレス学習：** 重い紙の教科書やノートを捨てましょう。読む、答える、検証する、そのすべてを 1 つのアプリで完結させます。
- **AI による即時フィードバック：** 表示されている教科書の内容に基づいて、記述した回答に対する AI からの修正と解説をすぐに受け取れます。
- **連続スクロールと同期：** 読書ペインと回答シートが自動的に同期され、現在の位置を見失うことがありません。
- **複数の API プロバイダー：** OpenAI、DeepSeek、Ollama がプリセットされているほか、独自のカスタムエンドポイントも使用可能です。
- **多言語サポート：** インターフェースは英語、中国語、日本語、韓国語、スペイン語、フランス語、ドイツ語に対応しています。

## 実行

```bash
swift run StudyReaderMac
```

## macOS アプリとしてパッケージ化

```bash
sh Scripts/package-app.sh
open dist/StudyReaderMac.app
```

## 注意事項

- 本アプリは他のアプリのウィンドウをキャプチャしないため、「画面収録」の権限は必要ありません。
- DRM 保護された Kindle や Apple Books ファイルはサポートされていません。
- デフォルトのモデルは `gpt-4o` です（OpenAI 使用時）。API アカウントで別のモデルが必要な場合は、設定で変更してください。
