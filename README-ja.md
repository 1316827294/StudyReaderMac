# StudyReaderMac

左側でPDFまたはDRMなしのEPUBを読み、右側で回答を書き込み、現在の読書画面のスクリーンショットと表示されている回答テキストをOpenAIに送信してフィードバックを得る、ネイティブmacOS学習用リーダー。

## 実行

```bash
swift run StudyReaderMac
```

## macOS アプリとしてパッケージ化

```bash
sh Scripts/package-app.sh
open dist/StudyReaderMac.app
```

## 現在の MVP 機能

- PDFKit で PDF ファイルを開く。
- spine の XHTML を抽出し、WebKit でレンダリングすることで、一般的な DRM なしの EPUB ファイルを開く。
- 左右のスクロール位置を継続的に同期する。
- macOS キーチェーンに OpenAI API キーを保存する。
- 現在のリーダービューポートの画像と、現在表示されている回答テキストを OpenAI Responses API に送信する。
- 回答テキストと分析履歴を Application Support にローカル保存する。
- 保存されたセッションパスを使用して、最近開いた本をスタート画面に表示する。
- OpenAI の応答を、ユーザーのテキストを置き換えるのではなく、色付きの読み取り専用 Markdown フィードバックとして現在の回答の下に追加する。

## 注意事項

- 本アプリは他のアプリのウィンドウをキャプチャしないため、「画面収録」の権限は必要ありません。
- DRM 保護された Kindle / Apple Books ファイルはサポートされていません。
- デフォルトのモデルは `gpt-5.5` です。お使いの API アカウントで他のモデルが必要な場合は、設定で変更してください。
