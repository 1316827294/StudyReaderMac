# StudyReaderMac

macOS 原生学习阅读器，支持在左侧阅读 PDF 或无 DRM 的 EPUB，在右侧编写答案，并将当前阅读界面的截图和可见的答案文本发送到 OpenAI 获取反馈。

## 运行

```bash
swift run StudyReaderMac
```

## 打包为 macOS 应用

```bash
sh Scripts/package-app.sh
open dist/StudyReaderMac.app
```

## 当前 MVP 功能

- 使用 PDFKit 打开 PDF 文件。
- 通过提取书脊 XHTML 并在 WebKit 中渲染，打开常见的无 DRM EPUB 文件。
- 持续同步左右两侧的滚动位置。
- 将 OpenAI API 密钥存储在 macOS 钥匙串中。
- 将当前的阅读器视口截图以及当前可见的答案文本发送至 OpenAI Responses API。
- 将答案文本和分析历史本地保存在 Application Support 中。
- 使用保存的会话路径，在开始屏幕上显示最近打开的书籍。
- 将 OpenAI 的响应作为带有颜色的只读 Markdown 反馈，添加到当前答案下方，而不是替换用户的文本。

## 注意事项

- 本应用不会截取其他应用的窗口，因此不需要“屏幕录制”权限。
- 不支持受 DRM 保护的 Kindle 或 Apple Books 文件。
- 默认模型为 `gpt-5.5`；如果您的 API 账号需要其他模型，请在设置中进行更改。
